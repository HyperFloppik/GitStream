GitStream is a GitHub-to-CDN link converter built for developers who want instant access to 
repository files through multiple CDN and proxy mirrors. 

We support services such as:
1. jsDelivr
2. Statically
3. StaticDelivr
4. Githack
5. RawGit 
6. GithubRaw
